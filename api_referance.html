Vixen X API Referance

Introcudes you to our custom API with some functions including how it works.

string vix_get_clipboard(void)

Returns the clipboard’s current content

string vix_set_clipboard(string Content)

Sets the clipboard’s current content to ‘Content’

string vix_dumpstring(string Script)

Dumps the bytecode for the Script chunk. 

string vix_decompile(ModuleScript> Script, bool Bytecode)

Decompiles Script and returns the decompiled script, or its bytecode if Bytecode is true. 

bool vix_islclosure(Function Func)

Returns true if Fn is a Lua function.

bool vix_checkcaller(void)

Returns true if the current thread is owned by Vixen X.

function vix_newcclosure(function Fn)

Pushes a CClosure that invokes Fn. Used for metatable hooks.

function vix_hookfunction(function Target, function Hook)

Hooks Target and replaces it with Hook. Returns the unhooked function.

void vix_context_set(int Context)

Sets the current thread context to Context.

int vix_context_get(void)

Returns the current thread context as an integer.
